¡Bienvenido a Pythia!

¿Qué es Pythia?

Pythia es un proyecto iniciado por estudiantes de la carrera FullStack Developers, con el objetivo de aplicar los conocimientos adquiridos y a la vez compartirlo con otros. En Pythia creemos que una parte importante del proceso de aprendizaje es la práctica y la experiencia, por lo que aplicamos el concepto "Learn-by-doing".

¿Qué vas a encontrar en Pythia?

En Pythia vamos a compartir videos y tutoriales sobre desarrollo web FullStack. ¡Próximamente nos abriremos a otras áreas de conocimiento también! Por eso te animamos a que te suscribas a nuestro Newsletter. Allí iremos colocando las novedades más relevantes sobre los contenidos de nuestro sitio web y sobre tus áreas de interés.

Tengo dudas sobre este sitio y sus autores...

¡Te entendemos! Te invitamos a que veas nuestra sección "preguntas frequentes" (FAQ) para que veas si tus dudas tienen respuesta; de lo contrario podés llenar el formulario para que nuestro staff te responda.

Estos son algunos de los cursos que encontrarás en nuestro sitio:

HTML

"Hyper-Text Markup Language", es un sistema de etiquetas que nos permite crear el contenido de una página web. Conforma "el esqueleto" de ellas.

CSS

"Cascading Style-Sheets", son documentos que nos permiten editar el estilo de nuestros documentos HTML. ¡Gracias a estas, nuestras páginas web serán mucho más atractivas!

JavaScript

A diferencia de HTML y CSS, JavaScript es un lenguaje de programación. Este acompaña los antemencionados, dándole a nuestras páginas funcionalidad.

GIT

GIT es un sistema de control de versiones. Nos permite llevar un registro de los cambios en los archivos de desarrollo, y facilitan el trabajo colaborativo.

Python

Python es un lenguaje de programación interpretado de alto nivel. A diferencia de otros lenguajes, Python hace hincapié en la legibilidad de su código.

SQL

"Structured Query Language" (Lenguaje de consulta estructurada), es un lenguaje diseñado para administrar y recuperar información de sistemas de gestión de bases de datos relacionales.
